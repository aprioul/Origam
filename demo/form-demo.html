Getting Started

Introduction

Origam contain an easy-to-use JavaScript library, which allows you forget about boring creating forms and problems associated with them. It does not require knowledge of JavaScript, so all parameters can be set as attributes of the input, textarea or select element.

Let's see a simple code needed to create a field


<input class="text-field--group__input" data-form="input" type="text" id="Text"/>

Install

There is no need to install any software. Simply place the appropriate scripts and styles in the <head> section.


<head>

    <link href="formify/css/default/origam.min.css" rel="stylesheet" >

    <script src="formify/origam.min.js"></script>

</head>

Styling

Each element is converted according to the same pattern, so creating your own style is very simple. The pattern looks like this:


<div class="text-field">

    <label class="text-field--label" for="Text">Text</label>

    <div class="text-field--group">

        <input class="text-field--group__input" data-form="input" type="text" id="Text"/>

    </div>

</div>
